CSS Media Queries

How Does a CSS Media Query Work, and Why Is It Useful?

A CSS media query is a rule that applies CSS styles only if certain conditions about the user's device or screen are true. Media queries allow web pages to adapt their layout and appearance for different devices, such as phones, tablets, and desktops. This is the foundation of responsive web design, making websites more usable and visually appealing on all screen sizes.

Media queries use the @media rule, followed by one or more conditions (called "media features"). If the conditions are met, the enclosed CSS is applied. For example, you can change font sizes, hide or show elements, or adjust layouts based on the screen width.

Common Media Features in Media Queries

Media features are the conditions you can test for in a media query. Here are four commonly used media features:

  • max-width: Checks if the viewport width is less than or equal to a certain value. Useful for applying styles to smaller screens.
    @media (max-width: 800px) { ... }
  • orientation: Checks if the device is in "portrait" (vertical) or "landscape" (horizontal) mode.
    @media (orientation: landscape) { ... }
  • prefers-color-scheme: Detects if the user prefers a light or dark color theme.
    @media (prefers-color-scheme: dark) { ... }
  • resolution: Checks the pixel density of the screen, useful for targeting high-DPI (Retina) displays.
    @media (min-resolution: 2dppx) { ... }

Summary Table of Media Features

Media Feature Example Description
max-width @media (max-width: 600px) Applies styles when viewport is 600px wide or less
orientation @media (orientation: portrait) Applies styles in portrait mode
prefers-color-scheme @media (prefers-color-scheme: dark) Applies styles if user prefers dark mode
resolution @media (min-resolution: 2dppx) Targets high-resolution (Retina) screens
