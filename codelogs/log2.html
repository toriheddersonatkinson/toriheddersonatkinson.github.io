CSS Measuring Units

Absolute vs. Relative Units

In CSS, length units are used to size elements, set margins, padding, and more. These units fall into two main categories: absolute and relative.
Absolute units have a fixed size regardless of context. Examples include px (pixels), cm (centimeters), and in (inches). They are useful when you need precise control, but may not adapt well to different screens or devices.
Relative units depend on other measurements, such as the size of the parent element, the viewport, or the font size. Examples include em, rem, %, vw, and vh. These units help create responsive designs that adjust to different devices and user settings.

Selected CSS Units Explained

1. px, pt

  • px (pixels): Represents a single dot on the screen. It is an absolute unit in CSS, but its actual size can vary depending on the device's pixel density. Commonly used for precise layouts.
  • pt (points): Traditionally used in print, where 1pt = 1/72 of an inch. In web design, it's less common, but sometimes used for font sizes.

2. em, rem

  • em: Relative to the font-size of the element. For example, if the parent font-size is 16px, 2em equals 32px. Useful for scalable layouts.
  • rem: Relative to the font-size of the root (<html>) element. This makes it easier to maintain consistent sizing across a site.

3. vw, vh

  • vw (viewport width): 1vw is 1% of the width of the browser window. Useful for responsive layouts.
  • vh (viewport height): 1vh is 1% of the height of the browser window.
