CSS Color

How CSS Uses Color

CSS allows you to control the color of text, backgrounds, borders, and many other elements on a web page. Color is a fundamental part of web design, helping to create visual interest, highlight important content, and improve readability.

Key CSS Color Properties

  • color: Sets the color of the text inside an element.
    p { color: blue; }
  • background-color: Sets the background color of an element.
    div { background-color: #f0f0f0; }
  • border-color: Sets the color of the border around an element.
    span { border: 1px solid red; border-color: red; }

Ways to Specify Color in CSS

CSS accepts color values in several formats. Here are some of the most common:

1. Named Colors

CSS supports over 140 named colors, such as red, blue, goldenrod, and mediumseagreen.

2. Hexadecimal Colors

Hex codes start with # and are followed by 3 or 6 hexadecimal digits. For example, #FF5733 is a shade of orange.

3. RGB Colors

RGB stands for Red, Green, Blue. You can specify values from 0 to 255 for each color channel.

4. HSL Colors

HSL stands for Hue, Saturation, and Lightness. This format is often easier to adjust for designers.
